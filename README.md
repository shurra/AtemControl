# AtemControl

Arduino based BlackMagicDesign switcher controller.

## Getting Started


### Prerequisites
1. Libraries:
	- https://github.com/kasperskaarhoj/SKAARHOJ-Open-Engineering/tree/master/ArduinoLibs
	- VirtualWire
2. Hardware:
	- Arduino Mega
	- PS/2 keyboard
	- 433MHz transmitter
	- 433MHz Superheterodyne Receiver
